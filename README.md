# なんでも 伝 言 板
- 昔はどこの駅にでもあった、黒板の伝言板をイメージしました。
- 色は二色に抑え、表示も少なめ、なるべくシンプルにしました。 「投稿」 は 「押」、「削除」 は 「消」 です。
- 見える部分以外のコードはN予備校 プログラミング入門コース 第3章「秘密の匿名掲示板」そのままです。
- (改行の機能はありません)
